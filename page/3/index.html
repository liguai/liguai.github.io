Unity通过for循环批量注册button事件，数组越界异常。
发表于2022-04-16|Unity•编程航线
for循环给button动态添加事件的过程中，会报出数组越界的情况。 如 12345for (int i = 0; i < 12; i++) { cardItem[i].iconImg.sprite = Resources.Load("Sprites/IconImage/" + index, typeof(Sprite)) as Sprite; cardItem[i].button.onClick.AddListener(() => OnbuttonClick(i)); } 结果sprite赋值是可以执行到的，但是button.onClick.AddListener却出现下标越界。 查阅资料后发现，for循环使用的参数是公用参数，我理解为即i的指针地址为同一个。这样对每一个button传参的时候传的都是同一个地址。 所以批量注册button事件的方式就需要添加一个另外开辟一个内存来记录传参值。
